## Reporting a Vulnerability

Jika error masih muncul, gunakan command
```
flutter clean
```
untuk menghapus semua dependencies yang ada, lalu jalankan lagi command 
```
flutter pub get
```
Untuk mendapatkan dependencies terbaru
